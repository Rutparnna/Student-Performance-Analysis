🎓 Student Performance Analysis

📌 Project Overview

This project analyzes student academic performance using Python. The objective is to explore the dataset, clean the data, perform statistical analysis, and visualize important insights related to student grades and study habits.

📊 Dataset

- Dataset Name: Student Performance Dataset
- File: "student-mat.csv"
- Source: UCI Machine Learning Repository

🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab / Jupyter Notebook

📈 Project Tasks

- Imported and explored the dataset
- Cleaned the data and checked for missing values
- Performed descriptive statistical analysis
- Calculated the average final grade (G3)
- Identified students scoring above 15 marks
- Analyzed the relationship between study time and final grades
- Compared academic performance based on gender
- Created data visualizations including:
  - Histogram of final grades
  - Scatter plot of study time vs. final grade
  - Bar chart of average grades by gender

📷 Output

The notebook includes tables, statistical summaries, and graphical visualizations that help understand factors affecting student performance.

🚀 How to Run

1. Open the notebook in Google Colab or Jupyter Notebook.
2. Upload the "student-mat.csv" dataset.
3. Run all notebook cells sequentially.
4. View the analysis results and graphs.
